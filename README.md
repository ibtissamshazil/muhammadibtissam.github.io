## ğŸ“ Adding Images

### Required Images

1. **Profile Photo**
   - Path: `media/profile.jpg`
   - Recommended: 400x400px, square format

2. **Company Logos** (in `media/logos/`)
   - `aimfit.png`
   - `carswitch.png`
   - `campus-kaizen.png`
   - `curemd.png`
   - `dgtx.png`
   - `lums.png`
   - Recommended: 200x200px, transparent background preferred

3. **Banner Images** (for experience cards)
   - `media/carswitch/banner.jpg`
   - `media/dgtx/banner.jpg`
   - `media/campus-kaizen/banner.jpg`
   - `media/aimfit/banner.jpg`
   - `media/lums/banner.jpg`
   - `media/curemd/banner.jpg`
   - Recommended: 1200x200px, landscape format

4. **Case Study Screenshots**
   - `media/aimfit/web-1.jpg`, `web-2.jpg`
   - `media/aimfit/mobile-1.jpg`, `mobile-2.jpg`
   - `media/carswitch/web-1.jpg`, `web-2.jpg`
   - `media/carswitch/mobile-1.jpg`, `mobile-2.jpg`
   - Recommended: Web (1200x800px), Mobile (400x800px)

### Image Tips

- Use high-quality images (but optimize file sizes)
- Maintain consistent aspect ratios
- Use JPG for photos, PNG for logos
- Compress images before uploading (use tools like TinyPNG)
